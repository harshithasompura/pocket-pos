# PocketPOS Screenshots

These are real PocketPOS captures from an iPhone 15 simulator at 1179 × 2556:

- `splash.png` — PocketPOS launch screen
- `setup.png` — first-run business setup
- `inventory.png` — catalogue with three demo products and a low-stock warning
- `product-detail.png` — Masala Tea stock controls and opening-stock history
- `sell.png` — searchable product catalogue and new-bill cart entry point
- `bill-detail.png` — completed offline invoice with item snapshots, discount, payment, and totals
- `receipt-actions.png` — completed invoice with durable print status, system printing, and PDF sharing actions
- `dashboard.png` — Today-range sales, bill, unit, payment, and low-stock analytics from SQLite

The demo catalogue contains Masala Tea, Filter Coffee, and Paper Bag. The completed invoice and dashboard use the same simulator SQLite database and invoice `INV-000001`; no mockup or generated UI was substituted.

Expo Go's development-tool button can appear in development captures. Receipt actions were visually verified in the iPhone 15 simulator; system print/share sheets and thermal-paper sizing require the final Android device and printer check. Final Android release captures can replace these images after the APK milestone without changing the README layout.
